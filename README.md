# Kulüp Kitaplığı

Okul kulübü için vintage tarzında tasarlanmış web kütüphane uygulaması.

## 🎯 Özellikler

- Kitap arama ve filtreleme
- Favorilere ekleme/çıkarma  
- Kategoriye göre filtreleme
- Yerel depolama (localStorage)
